# 宜蘭生日小旅行 2026

此專案可直接部署到 GitHub Pages。

## 部署方式

1. 在 GitHub 建立新的 Repository，建議名稱：`yilan-trip-2026`
2. 將本資料夾內的全部檔案上傳到 Repository 根目錄
3. 進入 Repository：`Settings` → `Pages`
4. `Build and deployment` 選擇：
   - Source：`Deploy from a branch`
   - Branch：`main`
   - Folder：`/ (root)`
5. 按下 `Save`
6. 等待約 1–3 分鐘後，網站網址會顯示在 GitHub Pages 頁面

網址格式通常為：

`https://你的GitHub帳號.github.io/yilan-trip-2026/`

## 注意

- 即時天氣需要網路連線
- 出發前清單使用瀏覽器 localStorage，只會保存在目前裝置與瀏覽器
- 網站內容若放在公開 Repository，任何知道網址的人都可開啟
